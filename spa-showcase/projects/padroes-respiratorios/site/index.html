Despertar Lúcido
Para manhãs conscientes e energizadas
Tranquilidade (3-0-6-0) • 4min
→
Energia Calma (5-0-2-0) • 4min
Começa com limpeza parassimpática profunda através de exalações prolongadas, removendo a inércia do sono. A transição para ativação simpática controlada cria um despertar simultaneamente suave e energizante.
Foco Laser
Para trabalho profundo e concentração intensa
Box Plus (6-6-6-6) • 5min
→
Tactical (5.5-5.5-5.5-0) • 3min
A simetria completa estabelece coerência cardíaca profunda. A transição para simetria parcial mantém a coerência criando fluxo mais dinâmico, ideal para períodos de concentração sustentada.
Reset Rápido
Para transições eficientes entre tarefas
Melhor Ventilação (3-0-8-1.8) • 3min
→
Box Plus (6-6-6-6) • 3min
A ventilação profunda "varre" padrões neurais da tarefa anterior, enquanto Box Plus estabelece novo estado de equilíbrio. Previne fadiga de decisão ao longo do dia.
Pré-Meditação Express
Preparação rápida para práticas contemplativas
Sossega Leão (4-7-8-0) • 5min
→
Endurance (5-0-8-5) • 5min
Cria base de calma através de retenção com pulmões cheios, depois aprofunda com retenções vazias. Treina rapidamente tolerância respiratória facilitando entrada em estados meditativos profundos.
Recuperação Atlética
Otimização pós-exercício
Energia Calma (5-0-2-0) • 2min
→
Sossega Leão (4-7-8-0) • 6min
Inicia com leve ativação para prevenir queda abrupta de catecolaminas. A transição gradual para recuperação profunda evita o "crash" pós-exercício, facilitando processos regenerativos.
Modo Competitivo
Para gaming, apresentações ou performances
Tactical (5.5-5.5-5.5-0) • 4min
→
Energia Calma (5-0-2-0) • 2min
Estabelece linha de base de alerta calmo, depois adiciona camada extra de ativação nos minutos finais. Cria o estado ideal de "estar na zona" - reflexos rápidos com controle emocional.

Princípios Fundamentais
Transições Graduais: Use padrões intermediários entre técnicas muito diferentes
Memória Respiratória: Efeitos persistem 3-5 minutos após mudança de padrão
Ordem Sequencial: Energização → Foco → Relaxamento (não o inverso)
Duração Mínima: 3-4 minutos por padrão para consolidar efeitos
Respiração Nasal: Padrão universal (exceto técnicas específicas)
Base Diafragmática: Fundamental para todos os padrões
Indicadores de Transição
Frequência cardíaca estabilizada indica momento ideal para mudança
Sensação de "saciedade respiratória" sinaliza prontidão para novo padrão
Mudança espontânea no ritmo sugere necessidade de ajuste
Qualidade da atenção (dispersão/hiperfoco) guia escolha do próximo padrão
